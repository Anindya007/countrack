countrack.me

Backup your countrack.me data.

Countrack.me is a 100% JavaScript app without any server side code. Which also means that it doesn't store your personal information in server and you are in full control of where and how you store the information you track with countrack.me.

As of now, Countrack.me is integrated with Firebase so that you can backup your data and use it across multiple devices.

Steps to create a Firebase URL

Firebase is a scalable, real-time backend for web applications. You can create a free Firebase account and use it with countrack.me to store your data.

Follow the steps below to create your Firebase URL and configure it with countrack.me.

  1. Go to https://www.firebase.com
  2. Click Signup and provide the required information.
  3. Provide a meaninful name when it asks to Create a Firebase. For example, you can use your name. Remember this URL as we need this to configure it with countrack.me.

Configure Firebase URL with Countrack.me

Now that you have created your own Firebase URL, you need to tell Countrack.me to use it as a data store.

  1. Go to http://countrack.me.
  2. Click / tap on the menu icon on the top right of the application.
  3. Enter the complete Firebase URL and click 'save'.

Thats it. After you have configure Firebase with Countrack.me, you can export / import your data.
